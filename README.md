## 図鑑データ
`dex/` 以下に同梱済みです。外部サイトに取りに行きません。

## 保存/読込
上部の「保存（JSON）」で構築を保存できます。読込で復元できます。

## 簡易シミュ
- それぞれのチームから3体をチェックして「選出でシミュする」
- タイプ相性 + 入力した技タイプを元に「ざっくり」評価します（本気の対戦エンジンではありません）

## おまかせ選出
- 「左をおまかせ選出」「右をおまかせ選出」「両方おまかせ」で、6体から3体を自動でチェックできます。
- いまの簡易評価（タイプ相性＋技タイプ＋S補正）を使って、相手が不利になりにくい3体を想定します。
- 「相手選出を隠す」をONにすると、右側のチェック状態とシミュ表の相手名を隠します。


## レギュ（ポケモンのみ）
- 画面上部の「レギュCSV読込」でCSVを読み込み、「レギュ適用」をONにすると、TRUEのポケモンのみが候補になります。
- 便利機能: リポジトリのルート（index.htmlと同じ階層）に `regulation.csv` を置くと、
  「同梱レギュ読込」ボタン、または「レギュ適用」ON時に自動で読み込みを試みます。
- CSV形式（どれでもOK / ヘッダー推奨）:
  1) `id,allow`（Showdown形式ID）
     - `id`: 例 `scizor`, `dondozo`
     - `allow`: TRUE/1/yes/on の場合のみ採用
  2) `No,名前,allow` または `名前,No,allow`（図鑑番号で指定）
     - `No`: 全国図鑑番号（数字）
     - `名前`: 日本語名（フォルムを区別したい場合に入れると安全）
     - `allow`: TRUE/1/yes/on の場合のみ採用
  ※ヘッダー無しの場合も、1列目が数字なら図鑑番号として扱います。

例（id）:
```csv
id,allow
scizor,TRUE
dondozo,TRUE
```

例（図鑑番号+名前）:
```csv
No,名前,allow
212,ハッサム,TRUE
977,ヘイラッシャ,TRUE
987,ハバタクカミ,TRUE
```
